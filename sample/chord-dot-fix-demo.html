Problem Root Cause

  • SVG default transform-origin is (0,0), not element center
  • transform: scale() without proper origin causes position drift
  • Browser inconsistencies with SVG transform-origin support

Recommended Solution Benefits

  • No position shifting - elements stay in place
  • Better browser compatibility
  • Follows Apple HIG interaction patterns
  • Maintains accessibility standards
